# 💫 About Me

Software Engineer specializing in building **secure, scalable, and high-performance** web & mobile applications.

I work primarily with **Java, Flutter, C#, JavaScript, React, and Node.js**, focusing on clean architecture, system design, and maintainable codebases.

Alongside software development, I actively practice **Cybersecurity** through hands-on platforms like *TryHackMe*, strengthening my skills in vulnerability analysis, secure coding, and system hardening.

I am passionate about solving real-world problems with technology and continuously improving my expertise in modern backend systems, cloud technologies, and scalable software architecture.

> 🚀 Focused on building impactful, secure, and production-ready software solutions.
